# Antidote plugins
#
# Load order matters here. Constraints:
#  - fzf-tab must load AFTER compinit (it wires into compsys at load
#    time). zephyr's completion plugin runs compinit synchronously
#    when its `immediate` zstyle is set (see config.d/antidote.zsh).
#  - OMZ lib must load BEFORE fast-syntax-highlighting. OMZ's
#    lib/misc.zsh does `zle -N self-insert url-quote-magic` which
#    REPLACES (not wraps) whatever's bound to self-insert. If fsh
#    runs first, that binding wipes out fsh's self-insert wrapper
#    and typed text never triggers re-highlighting.
#  - fast-syntax-highlighting must load AFTER OMZ + fzf-tab.
#    Wraps the final widget set so highlighting fires on key events.
#  - zsh-autosuggestions must be LAST: it wraps the final ZLE widget
#    set, including fsh's already-wrapped versions.

romkatv/powerlevel10k

zsh-users/zsh-completions path:src kind:fpath
MichaelAquilina/zsh-you-should-use

# compinit runs synchronously here (zephyr `immediate` zstyle).
mattmc3/zephyr path:plugins/completion

# fzf-tab requires compinit to have run.
Aloxaf/fzf-tab

# OMZ — must come BEFORE fsh. OMZ's lib/misc.zsh binds self-insert
# to url-quote-magic, which would overwrite fsh's self-insert wrap
# if fsh loaded first.
getantidote/use-omz

# fsh wraps widgets at source time; must come AFTER OMZ + fzf-tab so
# the bindings (self-insert → url-quote-magic, fzf-tab-complete, etc.)
# are already created.
zdharma-continuum/fast-syntax-highlighting

# Must be last — wraps the final widget set from everything above.
