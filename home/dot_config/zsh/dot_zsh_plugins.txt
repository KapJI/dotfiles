# Antidote plugins
#
# Load order matters here. Constraints:
#  - zephyr's completion plugin DEFERS compinit to its post_zshrc
#    (precmd) hook — the `immediate` zstyle is deliberately NOT set,
#    because config.d/antidote.zsh patches run_compinit after the
#    bundle is sourced and the patch only takes effect if compinit
#    runs later. Plugins loading before compinit still work: zephyr
#    queues compdef calls and replays them when the real compinit
#    runs at the first prompt, and fzf-tab autoloads _main_complete
#    straight from fpath at source time. The first Tab press always
#    comes after the first prompt, so compsys is live by then.
#  - OMZ lib must load BEFORE fast-syntax-highlighting. OMZ's
#    lib/misc.zsh does `zle -N self-insert url-quote-magic` which
#    REPLACES (not wraps) whatever's bound to self-insert. If fsh
#    runs first, that binding wipes out fsh's self-insert wrapper
#    and typed text never triggers re-highlighting.
#  - fast-syntax-highlighting must load AFTER OMZ + fzf-tab.
#    Wraps the final widget set so highlighting fires on key events.
#  - zsh-autosuggestions must be LAST: it wraps the final ZLE widget
#    set, including fsh's already-wrapped versions.
